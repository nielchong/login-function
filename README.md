# Personal Website
Exercise in building a personal website.

Java for back-end. Using Spring Boot to build the login and logout functionality of the website, Spring Data JPA for connection to the database to retrieve username and passwords, and Spring Security for authentication.

React.js, HTML, CSS for front-end.

Start the React app by navigating to the directory ../personal-website/frontend and running npm start in the console. React is hosted on https://localhost:3000. Useful for editing the front-end. Run 'npm run build' from the front-end and shift the contents of the /build folder to /src/main/resources/static.

Find LoginApplication.java in the backend folder and run it to start Spring Boot. Build files for the front-end are already included here and is hosted on https://localhost:9001.

The database used to store usernames and passwords is MySQL.
